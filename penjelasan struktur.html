Mekanisme Perancangan Web

Berikut adalah beberapa simbol atau rumus yang sering digunakan dalam HTML:

  • < dan >: Digunakan untuk membuat tag. Contohnya <h1> adalah tag untuk membuat heading level 1.
  • </>: Digunakan untuk menutup tag. Contohnya </h1> adalah tag penutup dari <h1>.
  • #text: Digunakan untuk menentukan teks yang akan ditampilkan. Contohnya <h1>#text</h1> akan menampilkan teks "#text" sebagai heading level 1.
  • class dan id: Digunakan untuk menentukan kelas atau id pada tag. Contohnya <div class="container" id="main"> digunakan untuk menentukan kelas "container" dan id "main" pada tag <div>.

Struktur HTML

Struktur dasar dari sebuah dokumen HTML terdiri dari dua bagian utama, yaitu <head> dan <body>.

  • Bagian <head> digunakan untuk menentukan informasi tentang dokumen, seperti judul, encoding, dan meta data.
  • Bagian <body> digunakan untuk menentukan konten dari dokumen, seperti teks, gambar, dan video.
